# Support

For usage questions, first check the [README](README.md) and existing [issues](https://github.com/Maylisten/go-test-tree-toggle/issues).

If the problem persists, open a bug report and include your VS Code version, extension version, operating system, workspace type, relevant `files.exclude` settings, and reproduction steps. Do not include secrets or private source code.

Security vulnerabilities must be reported privately as described in [SECURITY.md](SECURITY.md).
